C++ Flow Control

Flow control statements let you execute code conditionally, easily repeat code multiple times, or some combination of the two: repeat code until some condition is met.

The if Statement

The simplest thing you can do is to have a block of code that only runs when some condition is met. Use the if statement for this:

#include <iostream>

int main() {
  int x;
  std::cin >> x;

  std::cout << "You said: " << x << "\n";
  if(x < 10) {
    std::cout << x << " is pretty small.\n";
  }

  return 0;
}

The if statement allows you to put an else statement after it. The code in the else block only runs if the code in the if block did not run:

#include <iostream>

int main() {
  int x;
  std::cin >> x;

  std::cout << "You said: " << x << "\n";
  if(x < 10) {
    std::cout << x << " is pretty small.\n";
  }
  else {
    std::cout << "Woah... I can't count that high.\n";
  }

  return 0;
}

You can chain multiple if ... else statements together to handle multiple conditions.

#include <iostream>

int main() {
  int x;
  std::cin >> x;

  std::cout << "You said: " << x << "\n";
  if(x == 3) {
    std::cout << "That's three!\n";
  }
  else if(x % 2 == 1) {
    std::cout << "That's some other odd number.\n";
  }
  else {
    std::cout << "I don't know this one...\n";
  }

  return 0;
}
