Functions

Basic Function Stuff

Functions are blocks of code have names and can be "called," allowing you to reuse the same piece of code while only writing it once. A function declaration has multiple parts: the return type of the function, its name, a list of the types and names of all its parameters (in parentheses), and the code block to run with the given parameters.

Functions that don't return anything should have the return type void. Functions that don't take any parameters can simply leave their parentheses empty.

#include <iostream>

int gcd(int a, int b) {
  while(a != 0) {
    int t = b % a;
    b = a;
    a = t;
  }

  return b;
}

void print_gcd(int a, int b) {
  std::cout << "gcd(" << a << ", " << b << ") = ";
  std::cout << gcd(a, b) << "\n";
}

void print_done() {
  std::cout << "And I'm outta here...\n";
}

int main() {
  int x;
  std::cout << "Enter a number: ";
  std::cin >> x;

  print_gcd(x, 12);
  print_gcd(x, 13);
  print_gcd(x, 14);
  print_gcd(x, 15);
  print_done();
  return 0;
}

Function parameters only exist inside the function's code block. The a and b in gcd() are different from the a and b in print_gcd(), even though they have the same names.

The return Statement

Functions with a non-void return type must have a return statement so the compiler knows what value to pass back to the caller. Void functions will automatically return when they reach the end of their code blocks. Any function can use return to return early.

#include <iostream>

bool is_prime(int n) {
  if(n < 2) {
    return false;
  }

  for(int f = 2; f < n; ++f) {
    if(n % f == 0) {
      return false;
    }
  }

  return true;
}

void print_primality(int x) {
  if(x == 13) {
    std::cout << "Error. Unlucky. Fail.\n";
    return;
  }

  if(is_prime(x)) {
    std::cout << x << " is prime.\n";
  }
  else {
    std::cout << x << " is not prime.\n";
  }
}

int main() {
  int x;
  std::cout << "Enter a number: ";
  std::cin >> x;

  print_primality(x);
  return 0;
}

Common Mistakes

You can't call a function that the compiler doesn't know about, and the compiler reads your code from top to bottom. If you define a function after your main() function, you can't use it in main(), because the compiler hasn't seen it yet. Either define all your functions above main() or declare them in advance like this:

#include <iostream>

void gerbil(int n);

int main() {
  int x;
  std::cout << "How many gerbils are there? ";
  std::cin >> x;

  gerbil(x);
  return 0;
}

void gerbil(int n) {
  for(int i = 0; i < n; ++i) {
    std::cout << "GERBIL!\n";
  }
}
